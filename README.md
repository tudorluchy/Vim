<h1>My Vim<h1>

<h4>Installation<h4>

Just rename vim folder into .vim and vimrc file into .vimrc and drop then in $HOME.

Useful : If you use Vim on a server (no GUI), open .vimrc and comment line "colorscheme symfony" and "set t_Co=256"
